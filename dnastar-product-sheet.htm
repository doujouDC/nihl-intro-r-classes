DNASTAR Lasergene

Description

DNASTAR Lasergene package includes all the applications in DNASTAR Lasergene Molecular Biology, Lasergene Protein, and Lasergene Genomics. The DNASTAR Lasergene is a suite of tools that contains the following workflows, which can be accessed via the DNAStar Navigator:

  • Molecular biology
  • Genomics
  • Transcriptomics:
  • Protein analysis ad modeling

Detailed Information about each workflow is included in a separate handout.

How to Access

The NIH Library licenses are available to all NIH researchers, that don’t have access through their IC or lab. If you are part of NCI, please contact the BTEP for more information. The library has two methods for accessing Lasergene software:

Library use of the software which is installed on one of our bioinformatics computers (Bioinformatics 2)

Access from your computer using one of our network licenses

Getting Help

Each DNASTAR Lasergene application is well documented with help pages and tutorials. See the website tutorial page for more information.
