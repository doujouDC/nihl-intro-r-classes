DNASTAR Lasergene

Description

DNASTAR Lasergene is sequence analysis software for molecular biology research, protein analysis, genomics, and transcriptomics studies. DNASTAR Lasergene contains the following workflows, which can be accessed via the DNAStar Navigator:

  • Molecular biology
  • Genomics
  • Transcriptomics:
  • Protein analysis ad modeling

Detailed Information about each workflow is included in a separate handout.

How to Access

The NIH Library licenses are available to all NIH researchers, that don't have access through their IC or lab. If you are part of NCI, please contact the BTEP for more information. The library has two methods for accessing Lasergene software:

  • Bioinformatics 2, one of our high-powered bioinformatics computers (NIH Library)
  • Network licenses (From your own computer)s

Getting Help

Each DNASTAR Lasergene application is well documented with help pages and tutorials. See the website tutorial page for more information.
